# TAKUMI ORIGINALS

石谷 拓巳さんの自己紹介サイトを、動画配信サービス風に見せる静的サイトです。

## コンセプト

自己紹介を「人生シリーズ」として見せるポートフォリオサイト。
見る人が読み進めたくなるように、Home、History、Top Skills、Next Stage をエピソード形式で構成しています。

## ページ構成

- `index.html`: Home
- `growth.html`: Episode 1: History
- `strengths.html`: Episode 2: Top Skills
- `goals.html`: Episode 3: Next Stage

## 主な実装

- 左サイドナビ
- スマホ用ハンバーガーメニュー
- スクロールリビール
- KPIカウントアップ
- スキルバーアニメーション
- History の開閉式エピソード
- Top Skills の `VIEW SKILL` 開閉カード

## 公開方法

`self_intro_streaming_site` フォルダごとNetlifyへドラッグ&ドロップします。
